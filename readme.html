Documentación: VR-Visitas (plugin para WordPress)

Actualizado: 12/10/2007

Funcionalidad
  • Al instalar el plugin, se crean dos bases de datos vr_visitas y vr_spam que te permiten llevar un registro de las visitas a tu página.
  • Los resultados se muestran como: Visitas totales y Últimas 24 horas.
  • También muestra datos del Copyright del autor y la fecha de actualización de la página.
  • A través del panel de administración puedes consultar unas sencillas estadísticas y gestionar la base de datos.

Instrucciones de Instalación
  1. Descomprimir los archivos en la carpeta: wp-content/plugins/visitas/
  2. Activar el plugin en el Panel de Administración de WordPress.
  3. En el panel de administración Visitas/Opciones, actualizar los valores iniciales.
  4. Seguir las instrucciones de uso en COMO USARLO.
Instrucciones de Actualización
  1. Desactivar el plugin en el Panel de Administración de WordPress.
  2. Descomprimir y sobreescribir los archivos en la carpeta: wp-content/plugins/visitas/
  3. Activar nuevamente el plugin en el Panel de Administración de WordPress.
Instrucciones de Uso
Uso General
  1. Abrir el fichero wp-content/themes/<TU TEMA>/footer.php
  2. Buscar:
    <div id="footer">
         <p>
  3. Añadir el siguiente código:

    <?php if(function_exists('vr_copyright')) { vr_copyright(); } ?>


    El resultado será:
    © YYYY (Your name) - Actualizado: dd/mm/yy - Visitas totales: 1 - Últimas 24 horas: 1

    * Puedes seleccionar la información mostrada a través de una nueva opción.
    * También puedes ver los resultados en la barra lateral usando el widget visitas si tu tema lo permite.

  4. El fichero actualizado.html, contiene un webbot que se actualiza cada vez que se guarda el archivo.
    Este webbot almacena la fecha de actualización del fichero que se utiliza para mostrar la fecha de actualización de tu página.
  5. Dentro del menú de Administración de WordPress, se instala una nueva opción: Visitas, con distintas ventanas para el manejo del plugin.
    1. Visitas:
      Muestra una lista de las últimas visitas registradas.
    2. Añadir filtro:
      Permite añadir palabras que se usarán como filtro para borrar entradas de la base de datos.
    3. Borrar filtro:
      Borra las palabras de la tabla de filtros.
    4. Borrar SPAM:
      Elimina los registros de la base de datos correspondientes a las visitas de Robots y de los referentes que coincidan con tu lista de filtros.
    5. Estadísticas:
      Muestra un resumen (en %) de las entradas registradas: por navegador, por sistema operativo, por número de visitas y por referente.
    6. Opciones:
      Actualiza los valores de las opciones usadas en el plugin.
      Se explican en la propia página.
